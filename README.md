**ENG** |
ESX spectate optimized and ui redesign, [F11 to open]
press [G] to open player control menu, you can see:

- Ic name,
- Money,
- Bank_money,
- Black_money,
- Job,
- Steam name,
- Inventory, 
- Guns
- Licenses


**Install**
add pbo_spectate to your resource folder
start pbo_spectate in your server.cfg
